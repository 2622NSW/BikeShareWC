# BikeShareWC

An example of the combination of Bike Share data (from Washington DC) and R Code.

Source: Peter Spangler (2019) (http://www.storybench.org/exploring-bike-rental-behavior-using-r/)

Exploring bike rental behavior using R

Includes Day and Hour csv files from the UCI Machine Learning Repository 
(https://archive.ics.uci.edu/ml/datasets/bike+sharing+dataset) 

"This dataset contains the hourly and daily count of rental bikes between years 2011 and 2012 
in Capital bikeshare system with the corresponding weather and seasonal information."

Source of data: Hadi Fanaee-T
